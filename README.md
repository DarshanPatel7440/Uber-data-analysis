# Uber-data-analysis

🚀 **Uber Data Analysis Project | From Scratch to Advanced**

Excited to share my latest project where I explored and analyzed Uber ride data using **Python & Streamlit**!

🔗 **Live App:** https://uber-data-analysis-by-stmlit.streamlit.app/

💡 **What this project is about:**
I built an interactive dashboard that transforms raw Uber data into meaningful insights — helping understand ride patterns, demand trends, and user behavior.

🛠️ **Tech Stack Used:**

* Python
* Pandas
* NumPy
* Streamlit

📊 **Key Highlights:**

* Cleaned and processed raw datasets efficiently
* Performed exploratory data analysis (EDA)
* Built an interactive Streamlit dashboard
* Visualized key metrics like ride frequency, peak hours, and trends
* End-to-end project — from data preprocessing to deployment

🎯 **What I learned:**

* Handling real-world messy data
* Building interactive data apps
* Turning data into actionable insights
* Deploying projects for public use

✨ This project helped me strengthen my skills in **data analysis, visualization, and storytelling with data**.
